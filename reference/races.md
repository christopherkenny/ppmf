# Race Classifications

This data includes the basic race classifications used for redistricting
to get to an easier to work with set of values. This does not include
`hisp` grouping which is controlled separately by race within the census

## Usage

``` r
data('races')
```

## Value

tibble with three columns

- code: the two digit code used to code races

- desc: the description of the races

- group: the summary group used

## Examples

``` r
data('races')
```
