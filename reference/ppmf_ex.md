# Example PPMF Data

Includes Perry County, Alabama PPMF data from the April 28, 2021 PPMF
data release. This is a subset taken from the 12-2 P data.

As each observation is a person, this does not cover every block in the
county and due to DAS, not every block with population appears in this
data.

## Usage

``` r
data('ppmf_ex')
```

## Value

tibble with sample ppmf data

## Examples

``` r
data('ppmf_ex')
```
